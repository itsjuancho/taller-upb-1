# Taller 1: UPB API REST

Taller para materia Desarrollo de Aplicaciones WEB - UPB MisionTIC.

### Utilizar código

Pasos para probar el código:

```bash
    $ git clone https://github.com/itsjuancho/upb-taller-apirest.git
    $ cd upb-taller-apirest/
    $ npm install
    $ npm run dev
```
